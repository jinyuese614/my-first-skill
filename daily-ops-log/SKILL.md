# YouTube 일일 운영일지 생성

YouTube 채널 성과 데이터를 입력받아 콘텐츠별 분석, 인사이트, 그리고 **Next Action**을 포함한 일일 운영일지를 자동 생성합니다.

## 핵심 기능

**입력**: 크리에이터별 YouTube 채널 성과 데이터
**출력**: 구조화된 운영일지 (Lark 호환 마크다운) + Next Action 체크리스트

## 대화 흐름

### 1. 초기 요청 처리

사용자가 스킬을 트리거하면, 다음과 같이 데이터를 요청합니다:

```
오늘 YouTube 운영일지를 작성할게요! 아래 정보를 알려주세요:

1. 오늘 업로드한 영상 (크리에이터별, 영상 제목)
2. 채널별 주요 성과 (조회수, 좋아요, 댓글, 구독자 증가 등)

자유롭게 텍스트로 알려주셔도 되고, 표로 붙여넣으셔도 돼요!
```

### 2. 입력 데이터 처리

**필수 정보**:
- 크리에이터명
- 영상 제목
- 핵심 지표: 조회수, 좋아요, 댓글

**선택 정보**:
- 영상 길이
- 시청지속률
- 썸네일 클릭률 (CTR)
- 구독자 증가
- 특이사항
- 이전 데이터 대비 변화

**입력 형식**: 자유 형식 텍스트 또는 표 형식 모두 수용

### 3. 예외 상황 처리

#### 경우 1: 데이터 없이 트리거만 실행
```
YouTube 운영일지를 작성하려면 오늘의 성과 데이터가 필요해요!

이런 식으로 알려주시면 돼요:

크리에이터A "AI로 작곡하기"
- 조회수: 12500, 좋아요: 890, 댓글: 45, 구독자 증가: +120

크리에이터B "ChatGPT로 블로그 쓰기"
- 조회수: 8900, 좋아요: 560, 댓글: 32, 구독자 증가: +85

YouTube Studio에서 숫자 확인하신 후 알려주세요!
```

#### 경우 2: 일부 데이터만 제공
- 제공된 데이터로 초안을 먼저 작성
- 빠진 크리에이터를 명시적으로 안내
- 예: "크리에이터B의 데이터가 없습니다. 추가하시겠어요?"

#### 경우 3: 모호한 입력 ("오늘 성과 좋았어")
- 구체적인 숫자를 요청
- 단, 정성적 코멘트는 "종합 인사이트" 섹션에 반영

## 출력 형식

다음 6개 섹션으로 구성된 마크다운 문서를 생성합니다:

```markdown
# 📊 YouTube 일일 운영일지 — YYYY-MM-DD

## 오늘 업로드한 콘텐츠

| 크리에이터 | 콘텐츠 제목 | 영상 길이 |
|-----------|------------|----------|
| [이름] | [제목] | [시간] |

## 채널별 성과 요약

| 크리에이터 | 조회수 | 좋아요 | 댓글 | 구독자 증가 | 특이사항 |
|-----------|--------|--------|------|-----------|---------|
| [이름] | [수치] | [수치] | [수치] | [수치] | [메모] |

## 콘텐츠별 분석

### [크리에이터명] — "[콘텐츠 제목]"
- ✅ 강점: [분석 내용]
- ⚠️ 약점: [분석 내용]
- 💡 인사이트: [분석 내용]

## 종합 인사이트

- [크로스크리에이터 패턴 1]
- [크로스크리에이터 패턴 2]
- [전반적 트렌드]

## 🎯 Next Action

### 즉시 실행 (24시간 이내)
- [ ] [구체적이고 즉시 실행 가능한 액션]
- [ ] [구체적이고 즉시 실행 가능한 액션]

### 단기 실행 (이번 주)
- [ ] [이번 주 내 완료해야 할 액션]
- [ ] [이번 주 내 완료해야 할 액션]

### 중장기 모니터링
- [ ] [지속적으로 추적해야 할 항목]
- [ ] [지속적으로 추적해야 할 항목]
```

### 출력 요구사항

1. **날짜 자동 포함**: 오늘 날짜를 YYYY-MM-DD 형식으로 헤더에 자동 삽입
2. **Lark 호환성**: 깔끔한 마크다운 문법 사용, Lark에서 깨지지 않는 형식
3. **테이블 포맷 유지**: 정렬이 올바르게 유지되도록 작성
4. **이모지 일관성**: 강점 ✅, 약점 ⚠️, 인사이트 💡, Next Action 🎯 사용
5. **Next Action 필수**: 반드시 즉시/단기/중장기 섹션 모두 포함

## 분석 가이드라인

### 콘텐츠별 분석 작성 시

**강점 파악**:
- 높은 조회수 대비 좋아요/댓글 비율
- 평균 대비 우수한 지표
- 도입부 후킹, 시청지속률 등

**약점 파악**:
- 낮은 engagement rate
- 이탈률 상승 구간
- CTA 효과성 부족

**인사이트 도출**:
- 콘텐츠 주제의 수요 확인
- 시리즈화 가능성
- 개선 방향 제안

### 종합 인사이트 작성 시

**크로스크리에이터 비교**:
- 크리에이터별 성장세
- 콘텐츠 스타일별 효과

**전반적 트렌드**:
- 오늘의 주요 발견
- 주목할 만한 변화

### Next Action 작성 시

**즉시 실행 (24시간 이내)**:
- 당장 조치 가능한 구체적인 액션
- 예: CTA 위치 조정, 급상승 요인 분석 회의 예약

**단기 실행 (이번 주)**:
- 이번 주 내 기획/실행해야 할 항목
- 예: 시리즈 기획안 작성, A/B 테스트 계획 수립

**중장기 모니터링**:
- 지속적으로 추적해야 할 트렌드나 지표
- 예: 성장 트렌드 주간 추적, 콘텐츠 패턴 월간 분석

## 범위

### 하는 것
1. 입력된 YouTube 성과 데이터를 정해진 양식의 운영일지로 구조화
2. 콘텐츠별 장단점 분석 및 개선 포인트 제시
3. 크로스크리에이터 비교 인사이트 도출
4. **즉시/단기/중장기 Next Action 자동 생성** (체크리스트 형태)

### 하지 않는 것
1. YouTube API에서 직접 성과 데이터 수집 (사용자가 직접 입력)
2. Lark 문서에 자동 업로드 (사용자가 복사/붙여넣기)

## 작성 원칙

1. **데이터 기반**: 제공된 숫자를 근거로 분석
2. **실행 가능**: 구체적이고 적용 가능한 인사이트와 Next Action
3. **간결성**: 각 섹션을 명확하고 간결하게 작성
4. **일관성**: 동일한 형식과 스타일 유지

## 완료 후 안내

운영일지 생성 완료 시 다음과 같이 안내:

```
Lark에 복사하셨나요? 수정이 필요한 부분이 있으면 알려주세요!
```
