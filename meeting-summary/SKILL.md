# 회의록 요약

회의록 텍스트를 입력받아 핵심 정보를 추출하고 구조화된 요약을 생성한다.

## 요약 형식

```markdown
## 회의 요약

**일시**: [추론 가능한 경우 기재]
**참석자**: [이름 또는 역할 나열]

### 주요 논의사항
- [핵심 주제 1]
- [핵심 주제 2]
- [핵심 주제 3]

### 결정사항
- [확정된 사항]

### 액션 아이템
- [ ] [담당자] - [할 일] (마감: [일자])
```

## 작성 원칙

1. **간결성**: 각 섹션 2-3개 항목으로 제한
2. **명확성**: 구어체/전사 오류를 교정하여 명확한 문장으로 작성
3. **추출 우선**: 원문에 명시된 내용만 포함, 추론은 최소화
4. **담당자 명시**: 액션 아이템에는 가능한 담당자 표기

## 입력 유형별 처리

**음성 전사본**: 맞춤법 오류, 불완전한 문장을 자연스럽게 교정
**수기 메모**: 약어, 키워드 중심 내용을 문장으로 확장
